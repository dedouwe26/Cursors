# Cursors
Cursors is a Minecraft mod and library that adds custom mouse cursors to GUIs.

## Cursors
There are 6 different cursors:
- Blocked: the hovered object is inaccessible.
- Default: There is nothing special about the hovered object.
- Grab: It is a slot you are hovering over.
- Grabbing: Your cursor is grabbing an item.
- Pointer: When the cursor points to something clickable.
- Text: Indicates a text input field.
